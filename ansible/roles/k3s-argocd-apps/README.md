k3s-argocd-apps
===============

Tasks to install argocd apps.

Requirements
------------

Debian based system.

Role Variables
--------------

Not at the moment.

Dependencies
------------

This role depends on k3s-master.

Example Playbook
----------------

To use this role, just include it in your playbook, for example:

    - hosts: servers
      roles:
         - k3s-argocd-apps
