k3s-agent
================

Install k3s agent on linux server

Requirements
------------

Debian based system.

Role Variables
--------------

These are default variables for this role:

* master_token: "random"

Dependencies
------------

Not at the moment.

Example Playbook
----------------

To use this role, just include it in your playbook, for example:

    - hosts: servers
      roles:
         - k3s-agent
