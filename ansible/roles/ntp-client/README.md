ntp-client
==========

Install ntp client on linux server.

Requirements
------------

Debian based system.

Role Variables
--------------

These are default variables for this role:

* ntp_server: "ntp.metas.ch"

Dependencies
------------

Not at the moment.

Example Playbook
----------------

To use this role, just include it in your playbook, for example:

    - hosts: servers
      roles:
         - ntp-client
