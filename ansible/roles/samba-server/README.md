samba-server
===========

Tasks for setting up local Samba CIFS service.

Requirements
------------

Ubuntu based system.

Role Variables
--------------

Dependencies
------------

Not at the moment.

Example Playbook
----------------

To use this role, just include it in your playbook, for example:

    - hosts: servers
      roles:
         - samba-server
